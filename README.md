# LegionFinder — American Legion Post Directory

**[LegionFinder.com](https://legionfinder.com)** is the most complete free directory of American Legion posts across all 50 US states.

## Features

- **5,000+ American Legion posts** listed and searchable
- - Browse by **state and city** with intuitive navigation
  - - **Contact info, ratings, and directions** for every post
    - - **Mobile-friendly** responsive design
      - - **Fast, static site** — loads instantly on any device
        - - **Schema.org markup** (LocalBusiness, BreadcrumbList) for rich search results
         
          - ## About the American Legion
         
          - The [American Legion](https://legionfinder.com/about-american-legion/) is the largest veterans organization in the United States, with nearly 13,000 posts worldwide. Founded in 1919 by veterans returning from World War I, the Legion advocates for veterans' rights, supports military families, and serves local communities through volunteer programs, youth activities, and patriotic observances.
         
          - ## How to Use
         
          - Visit **[legionfinder.com](https://legionfinder.com)** and browse by state, or search for a post near you. Each listing includes:
         
          - - Full address and phone number
            - - Google Maps directions
              - - Google ratings and reviews
                - - Official website link (when available)
                  - - Nearby posts in the same area
                   
                    - ## Technology
                   
                    - - Static HTML generated from a SQLite database of Google Places data
                      - - Hosted on Cloudflare Pages for global CDN delivery
                        - - JSON-LD structured data on every page
                          - - Comprehensive sitemap.xml with 5,000+ URLs
                            - - Security headers and caching via Cloudflare
